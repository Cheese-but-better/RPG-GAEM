# RPG-GAEM
A simple RPG gaem made with Python.

Note for new Python users:
To run the Python script, first navigate to the folder in which the file is stored. To do this, type "cd 'PATH/TO/YOUR/FOLDER'" and press enter.
Next make sure all of the required imports are installed. To do this, type "pip install IMPORT_NAME" and press enter, where IMPORT_NAME is the name of the import that you are downloading. You should download every import, of which there is a list at the top of the Python file.
Lastly, type "python3 RPG-main.py". Press enter, and the program should start.
